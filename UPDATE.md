# 一些比較大的更新
## 多工作區 [ Mulit WorkSpace ]
* 可以用 `Alt + W` 叫出來
* 每個工作區的桌布和主題色都是分開的
* 可以給每個工作區取名 或者寫個筆記讓自己知道這工作區是幹嘛的
## 執行 [ Run ]
* 可以用 `Alt + R` 或者 `F1` 叫出來
* 你可以用這個東西快速查標簽 查ID 開設定之類的
## 影片桌布 [ Video Wallpaper ]
* 啊 我們頭貼和桌布現在都可以用上影片了 真好
* 可是 有個問題 儘管我已經用上VideoMirror 但 我不知道爲什麽還是會卡到一些性能 啊...算了
* 然後還有一些奇奇怪怪的小問題我就不講了 就那樣
## 快速預覽 [ Peek Preview ]
* 你現在可以在pool或者postSearch裏面 對著任何的post 按下你的`Space` 他就會叫出一個叫做預覽的東西 放開就沒了
* 但我沒特別設計先預覽縮圖后全圖的這個設定 之後會改
* 然後 影片沒辦法看 拖到下一次更新
