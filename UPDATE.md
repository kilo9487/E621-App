# inDev 0.0.2 更新日誌

## 新增功能
- **語言切換器**：新增中英文語系切換功能。
- **開始選單拖曳**：開始選單內的項目現在支援拖曳（Drag）操作。
- **postSearch 篩選器**：在 postSearch 中新增了篩選系統（Filter）。
- **自訂大頭貼**：現在支援更改使用者的個人頭像。
- **複製 JSON**：現在可以複製 post 與 postSearch 的 JSON 資料。

### 新增快捷鍵：
- **postSearch**
  - `Shift + F`：開啟篩選器
  - `Shift + J`：跳轉頁面
- **桌面（Desktop）**
  - `Alt + ,`：最小化視窗
  - `Alt + →`：視窗貼齊右側
  - `Alt + ←`：視窗貼齊左側

## 問題修復
- 修復全螢幕視窗向下拖曳時，放開後視窗會回彈至原位的問題。
- 修復在使用觸控操作時，Viewer 檢視器的中心點會發生偏移的問題。
- 改善 postSearch 中 reload（重新載入）功能的操作體驗與穩定性。

## 已知問題
- 雖然 reload 功能已改善，但首次開啟 postSearch 時，仍有機率發生應用程式崩潰的問題。
  *（若遇到此情況，請透過開始選單開啟控制台檢查是否有錯誤訊息；若有報錯，請直接按 Data -> Reload 即可。）*

## 開發者筆記
- `_app.tsx`：沿用並移植了舊有程式碼，因此檔案中若出現部分目前未使用的功能屬於正常現象。
- `dragItem()`：新增了拖曳輔助函式（Helper）。開發者現在不需要再手動編寫 `JSON.stringify` 與 `e.dataTransfer.setData` 了。
- `e621Info()`：新增專用於獲取 e621 認證資訊的函式，獲取的資料可直接帶入 `user` 物件中使用。
- `menuBtn`：目前僅實作兩個按鈕：一個為複製 JSON 的按鈕，另一個為 post 相關按鈕。
